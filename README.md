Reconstrucción completa de una página de detalles de producto (PDP) de alto nivel, enfocada en la experiencia de usuario de marcas de lujo. Optimizada para ser increíblemente rápida y con animaciones perfectas.

### Características Principales

- **Arquitectura React + Vite**: Carga de componentes instantánea sin recargas de página.
- **Tipado Estricto (TypeScript)**: Evita errores comunes de estado en carritos de compra.
- **Micro-interacciones Premium**: Hover states, animaciones de botones y off-canvas menu diseñados en CSS puro.
- **Diseño Asimétrico y Tipografía**: Combinación de `Cormorant Garamond` y `Inter` para una estética editorial.
- **Carrito Off-Canvas Interactivo**: Control de cantidad y precio total calculado en tiempo real con React Hooks.
- **Despliegue Continuo (CI/CD)**: Configurado nativamente con GitHub Actions para publicar la versión optimizada en GitHub Pages automáticamente.

---

### Despliegue Local

Para correr este proyecto en tu entorno de desarrollo:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Luis-Manzanares/bouty-pdp.git
   ```
2. Entra al directorio:
   ```bash
   cd bouty-pdp
   ```
3. Instala todas las dependencias:
   ```bash
   npm install
   ```
4. Levanta el servidor local:
   ```bash
   npm run dev
   ```
